Instruções de Uso

Este é um chatbot para responder a dúvidas sobre a empresa Signa. Ele utiliza um modelo de IA que se baseia em informações do site da empresa.

Como Usar:

  1. Atualizar a Base de Conhecimento:
      Se a base de dados estiver vazia, clique em "Atualizar Base". O chatbot irá raspar o site da Signa e criar uma base de dados para conversação.
    1. Adicionar um Link Específico:
        Você pode inserir um link específico na caixa de texto e clicar em "Adicionar Link" para incluir somente essa página na base de conhecimento.
      1. Iniciar a Conversa:
          Após a base de dados ser atualizada, o chat será liberado e você poderá fazer perguntas sobre a empresa, seus produtos e serviços.
